Firebase Plugin Collection for Construct 3

Works on almost all Platforms

Designed to work particularly on Web, Android & iOS. (Partial support on Desktop Apps & Xbox UWP)

Guides & Examples

Comprehensive documentation and examples that provide step-by-step instructions, explanations, and best practices

Debugging Made Easier

Options to log detailed messages and errors into the Browser Developer Tools Console

Discord Community

Included Firebase Services

Plugins are built with these Firebase Services

*This product/service is not affiliated to Firebase in any way

Realtime-Database/Firestore

Create easy Chat Rooms, Advanced Lists, Manage Friends, Three-Event fully customizable Leaderboards & much more

Authentication

Create and manage User's Profiles, Sign In with Google, Facebook, Phone, Email, etc. and store data in user's account

Cloud Storage

Upload or Download user generated images or any file type in Cloud

Analytics

Track and get website traffic details

Remote Config

Modify appearance without relaesing update

Performance

Keep track of task and network performance

Crashlytics

Get crash information in Mobile apps
